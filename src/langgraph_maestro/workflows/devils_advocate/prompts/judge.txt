## Your Task
Evaluate both arguments fairly. For each disputed point, determine which side has the stronger argument and why.

Produce a verdict as a JSON object with these exact fields:
- "verdict": one of "PROCEED", "REVISE", "ABANDON"
- "confidence_score": float between 0.0 and 1.0 (how confident you are in this verdict)
- "reasoning": 2-3 paragraph explanation of your decision
- "strongest_critique_points": array of strings — the critique's best arguments
- "strongest_defense_points": array of strings — the defense's best arguments
- "conditions": array of strings — if PROCEED or REVISE, what conditions or changes are required

Verdict guidelines:
- PROCEED: Defense wins decisively. No fatal flaws. Risks are acceptable and mitigated.
- REVISE: Both sides have merit. Core proposal is sound but needs significant changes.
- ABANDON: Critique wins decisively. Fatal flaws are real and unmitigated. Better alternatives exist.

Return valid JSON only. No markdown fences or commentary outside the JSON.
